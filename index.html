Instrucciones

Evil Minion es muy escurridizo, derrotalo disparandole tantas veces sea necesario, demuestra que eres el mejor en este juego de habilidad y destreza.

  • Movimiento: Utiliza las teclas arriba o abajo para mover el Robot
  • Disparo: Para disparar presiona la tecla barra espaciadora ,debes esperar 2 segundos para volver a disparar.
  • Atención: Evil Minion puede cambiar de velocidad cada vez que suba y baje.
  • Limite: Tienes 1 minito para acertar, veamos que tan buena punteria tienes.
  • Iniciar Juego: Debes presionar iniciar juego en la parte inferior de la pantalla
  • Resultado: Este boton aparece en la parte inferior del juego, una vez que se ha terminado el tiempo, presionalo.
